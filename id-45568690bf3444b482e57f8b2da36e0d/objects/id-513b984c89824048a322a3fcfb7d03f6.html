Definities: Notificeren - Het door een aanbieder aan afnemers verstrekken van informatie over binnen zijn domein plaatsgevonden gebeurtenissen Notificatie - De door een aanbieder aan afnemers verstrekte informatie een binnen zijn domein plaatsgevonden gebeurtenis Typering van events: Signaal event = Signaal dat er iets is gebeurd - Sensor die een signaal geeft na een plaatsgevonden gebeurtenis Gegevensmutatie event = Oude en nieuwe gegevens na een verandering - Oude en nieuwe adres. Gegevensmutatie + Aanleiding event = Oude en nieuwe gegevens na een verandering + het type plaatsgevonden gebeurtenis - Oude en nieuwe adres = Verhuizing Informatierijk event = Gebeurtenisdata met inhoudelijke gegevens - Verhuizing van gezin met alle data i.p.v. van aparte berichten per gezinslid. Informatiearm event = Gebeurtenisdata zonder inhoudelijke gegevens (evt. apart op te halen). - Verhuizing met in bericht alleen identificerende gegevens. Te maken handreikingen: - handreiking voor welk type event wanneer geschikt is - handreiking voor ieder type event (met eisen, wensen en aandachtspunten) - handreiking over transitie van IST naar SOLL - Gebeurtenisgericht gaan werken (vanuit een gegevensgericht informatielandschap) - Definieren van gebeurtenistypes voor gebruik binnen events (op te nemen in NL GOV Profile) - Naamgeving van bronnen voor gebruik binnen events (op te nemen in NL GOV Profile) - Aandachtspunten bij inrichten van bronregistraties in relatie tot notificeren
